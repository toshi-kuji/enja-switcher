# EnJaSwitcher ウェブ配布

## 概要
EnJaSwitcher の紹介・ダウンロードページを Astro + Cloudflare Pages で作成し、無料で公開配布する。App Store は使わない。

---

## Phase 1: 配布戦略の決定

### 配布方法
サイトから直接ダウンロードできるようにする（ストア外配布、一般的な方法）。

### 署名方式
DL後にアプリを開けるかどうかに影響する。別途決定が必要。

| 方式 | コスト | DL後の手順 |
|------|--------|-----------|
| ad-hoc署名のみ（現状） | 無料 | `xattr -cr` または右クリック→開くが必要 |
| Apple Developer ID + Notarization | $99/年 | そのまま開ける |

※ ad-hoc署名はビルド時の技術要件（Apple Silicon対応、権限維持に必要）であり省略不可。
  ここでの選択は「Notarizationまでやるかどうか」。

- [ ] Notarization の要否を決定する
- Notarization なしの場合: macOS バージョンアップで回避策が塞がれるリスクあり
- Developer ID は後からでも移行可能

### 補助的な配布チャネル（任意）
- [ ] Homebrew Cask 対応するか決定する（quarantine 属性を Homebrew が処理するため Gatekeeper 警告なし）
- [ ] ソースからのビルド手順もサイトに掲載するか決定する

### ダウンロードファイルの置き場所

| 方式 | 特徴 |
|------|------|
| GitHub Releases | バージョン管理統合、2GB/file、無料 |
| CF Pages の public/ に同梱 | 最もシンプル、25MB制限（現状余裕） |
| Cloudflare R2 | カスタムドメイン可、無料枠10GB |

- [ ] 置き場所を決定する

### ドメイン
- CF Pages デフォルト: `enja-switcher.pages.dev`
- [ ] 独自ドメインを取得するか決定する

## Phase 3: サイト実装

### 技術構成
- Astro（静的出力 `output: 'static'`）
- Vanilla CSS のみ（React / Tailwind 不使用）
- JS フレームワーク依存なし（Astro コンポーネントのみ）
- 既存リポジトリ内 `website/` に配置

### 言語
- 日本語（`/`）+ 英語（`/en/`）
- JSON 辞書ファイルでテキスト管理

### ページ構成（シングルページ、日英共通構造）
1. ヒーロー: アプリ名、キャッチコピー、ダウンロードボタン、言語切替
2. 特徴: 2つの切り替え方式の説明
3. インストール手順: DL → Gatekeeper回避 → 起動 → 権限付与（ステップバイステップ）
4. 権限の説明: アクセシビリティ・入力監視が必要な理由、アプリが何をしていて何をしていないか
5. CapsLock方式の設定: macOS 側「Caps Lock → アクションなし」の手順
6. トラブルシューティング / FAQ
7. フッター: ライセンスページへのリンク、免責事項ページへのリンク

### ディレクトリ構成
```
website/
├── astro.config.mjs
├── package.json
├── public/
│   ├── favicon.ico
│   ├── og-image.png
│   └── screenshots/
└── src/
    ├── styles/
    │   └── global.css
    ├── layouts/
    │   └── Layout.astro
    ├── i18n/
    │   ├── ja.json
    │   └── en.json
    └── pages/
        ├── index.astro           # 日本語トップ
        ├── license.astro         # ライセンス（日本語）
        ├── disclaimer.astro      # 免責事項（日本語）
        └── en/
            ├── index.astro       # 英語トップ
            ├── license.astro     # License（英語）
            └── disclaimer.astro  # Disclaimer（英語）
```

## 注意点

- Gatekeeper: ad-hoc署名アプリは macOS が警告を出す → 回避手順を丁寧に記載
- 権限への不安: アクセシビリティ・入力監視を求める理由を明確に説明
- codesign と更新: バージョンアップ時の権限リセット手順もサイトで案内
- macOS 依存: `xattr -cr` や `CGEventTap` が将来使えなくなる可能性
- CF Pages 無料枠: ビルド500回/月、帯域・リクエスト無制限（十分）
